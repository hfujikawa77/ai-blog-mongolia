# 3/27号 ウランバートルを探検しよう！ 🏙️🦕🛕

## 1️⃣ ウランバートルってどんな街？

モンゴルの首都**ウランバートル（Улаанбаатар）**は、モンゴル語で「赤い英雄」という意味だよ。

草原や砂漠のイメージが強いモンゴルだけど、首都には高いビルが立ち並び、地下鉄も走るにぎやかな都市なんだ。
モンゴルの人口は約340万人で、そのうち約半分がウランバートルに集中しているよ。

| 基本データ | 内容 |
|------------|------|
| 正式名称 | ウランバートル（Улаанбаатар） |
| 意味 | 「赤い英雄」 |
| 人口 | 約160万人（2024年） |
| 標高 | 約1,350m（高地にある首都） |
| 冬の気温 | −30℃を下回ることも！ 世界最寒の首都のひとつ |

標高が高く、冬はとても寒い街だけど、夏はカラッと晴れた青空が広がって観光にぴったり。
草原と都市の両方を楽しめる、ちょっと不思議な魅力の街なんだ。

---

## 2️⃣ スフバートル広場でモンゴルの歴史を感じよう

ウランバートルの中心にある**スフバートル広場（Сүхбаатарын талбай）**は、街の心臓部だよ。

広場の名前は、1921年のモンゴル独立を率いたスフバートル将軍にちなんでいる。
広場の中央には将軍の騎馬像がそびえ立ち、その奥には国会議事堂がどっしりと構えているんだ。

国会議事堂の正面には、あの**チンギス・ハーン**の巨大な座像もある！
その左右には2人の将軍の像も並んでいて、まるでモンゴルの歴史が目の前に広がるようだよ。

| 見どころ | 説明 |
|----------|------|
| スフバートル騎馬像 | 広場中央にある独立の英雄の像 |
| チンギス・ハーン座像 | 国会議事堂正面の巨大な石造りの像 |
| 国会議事堂（モンゴル国大会議宮） | 重厚な外観のモンゴルの国会 |
| 広場のにぎわい | 休日は家族連れや観光客でいっぱい！ |

夜はライトアップされて、また別の顔を見せてくれるよ。

## 3️⃣ ガンダン寺院で仏教の世界へ

市内の小高い丘に建つ**ガンダン寺院（Гандантэгчинлэн хийд）**は、モンゴルで最も有名な仏教寺院だよ。

正式名称は「ガンダンテグチンレン寺院」で、「大いなる歓喜をもたらす寺院」という意味。
社会主義時代にいったん閉鎖されたけれど、1990年の民主化以降に再び開かれ、今では多くの人々が祈りをささげに訪れているんだ。

| 見どころ | 説明 |
|----------|------|
| ミグジット・ジャンライシグ像 | 高さ26.5mの黄金の観音菩薩像。寺院の本堂に鎮座する圧巻の像 |
| 僧侶たちのお勤め | 毎朝、境内で読経する僧侶の姿を見ることができる |
| 仏塔（ストゥーパ） | 敷地内に並ぶ白い仏塔は写真映えする！ |
| お土産マーケット | 境内周辺には仏教グッズや民芸品を売るお店が並ぶ |

靴を脱いでお堂に入ると、金色の大きな像が迎えてくれるよ。
静かな空間の中で、モンゴルの人々の信仰の深さを感じることができるんだ。

## 4️⃣ 恐竜博物館で古代モンゴルへタイムスリップ！

子どもにとっていちばんのお楽しみかもしれないのが、**モンゴル自然史博物館**の恐竜コーナーだよ！

モンゴルのゴビ砂漠は、世界有数の恐竜化石の産地。
ここでしか見られない貴重な恐竜の骨格標本が、ずらりと展示されているんだ。

| 展示ハイライト | 説明 |
|--------------|------|
| タルボサウルス | ゴビ砂漠で発見された大型の肉食恐竜。ティラノサウルスの仲間で迫力満点 |
| プロトケラトプス | 「フリル」と呼ばれる頭の飾りが特徴的な植物食恐竜。ゴビ砂漠産 |
| オヴィラプトル | 卵を抱いた状態で発見され、恐竜の育児行動の研究に貢献した恐竜 |
| 恐竜の卵の化石 | 実物の恐竜の卵の化石もある！大きさのひみつを探ろう |

「ゴビ砂漠がかつて恐竜だらけだった」なんて、なんともロマンがあるよね。
博物館では恐竜だけでなく、草原の動物・民族の歴史・天文学など、モンゴル自然のすべてが学べるんだ。

## 🗣️ モンゴル語でひとこと

| 日本語 | モンゴル語 | 読み方 | 🔊 |
|--------|------------|--------|----|
| 街・都市 | Хот | ホト | [🔊 発音](https://forvo.com/search/%D0%A5%D0%BE%D1%82/mn/) |
| 広場 | Талбай | タルバイ | [🔊 発音](https://forvo.com/search/%D0%A2%D0%B0%D0%BB%D0%B1%D0%B0%D0%B9/mn/) |
| 寺院・お寺 | Хийд | ヒード | [🔊 発音](https://forvo.com/search/%D0%A5%D0%B8%D0%B9%D0%B4/mn/) |
| 博物館 | Музей | ムゼイ | [🔊 発音](https://forvo.com/search/%D0%9C%D1%83%D0%B7%D0%B5%D0%B9/mn/) |

今回はウランバートルの街歩きにちなんだことばを選んだよ。
「Хот（ホト）」はシンプルに「街・都市」を意味する単語で、旅行中に「この街は大きいね！」と言うときに使えるよ。
「Музей（ムゼイ）」は日本語の「ミュージアム」とよく似ているね。モンゴル語にも外来語がたくさん使われているんだ！

## 🎲 今日のクイズ

Q: ウランバートルという名前の意味は何でしょう？

1. 青い空の都市
2. 赤い英雄
3. 草原の中心

<details>
<summary>答えを見る</summary>

**答え：2. 赤い英雄**

ウランバートル（Улаанбаатар）の「ウラーン（Улаан）」は「赤い」、「バータル（баатар）」は「英雄」という意味。3/25号でも紹介した「Баатар（英雄）」が首都の名前に入っているんだね。1921年のモンゴル革命を記念してつけられた名前なんだ。

</details>
